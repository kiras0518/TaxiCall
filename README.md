# TaxiCall

# 仿製 Uber 功能練習 

應用程序主要功能：
- Login/Signup page
- Logout
- Check rider request (disance, time, location)
- Get direction to the rider who call an Uber
- Get position of the driver in real time
  
應用程序使用：
- Framework Firebase
- MapKit
- Background location updates mode
- Navigation controller
- Table view controller
